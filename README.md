# Assignment 5: Quicksort Algorithm

## Overview

This project implements and analyzes two versions of the Quicksort algorithm:

* **Deterministic Quicksort** – Uses the last element as the pivot.
* **Randomized Quicksort** – Uses a randomly selected pivot.

## Files

```text
├── deterministic_quicksort.py
├── randomized_quicksort.py
├── README.md
└── report.pdf
```

## Requirements

* Python 3.x

## How to Run

Run Deterministic Quicksort:

```bash
python deterministic_quicksort.py
```

Run Randomized Quicksort:

```bash
python randomized_quicksort.py
```

## Summary

* Average Time Complexity: **O(n log n)**
* Worst Case: **O(n²)**
* Space Complexity: **O(log n)** (average)
* Randomized Quicksort reduces the likelihood of worst-case performance by selecting pivots randomly.
